The t-distribution & critical values

t 分配與臨界值表 · when σ is estimated, the ruler changes

Overview

When the population standard deviation is unknown and must be estimated from the sample, the standardised statistic no longer follows the normal curve but Student's t: heavier in the tails, more forgiving of extreme values. As the degrees of freedom grow, t approaches z. The critical-value table converts a chosen significance level α and degrees of freedom df into the cut-off a test statistic must clear.

Key formulas
$$t = \dfrac{\bar{X} - \mu}{S/\sqrt{n}}, \quad df = n - 1$$
$$t_{df} \to N(0,1) \ \text{as } df \to \infty$$
The rejection region

A critical value marks the edge of the rejection region: the tail area equal to α. A two-tailed test splits α between both tails; a one-tailed test puts it all in one. The shaded area below is exactly α, and its inner edge sits at the critical value the table returns.

Interactive critical-value table

Pick a df and α, or toggle one- vs two-tailed. The readout gives the critical value the test statistic must exceed. Click any df row or α column to select it.

How to read the table
  1. Find the row. Degrees of freedom: for a one-sample or paired test, df = n − 1; for an independent two-sample pooled test, df = n₁ + n₂ − 2.
  2. Find the column. Choose your α. For a two-tailed test the listed α is split across both tails; for a one-tailed test read the half-α column shown.
  3. Read the cell. That number is tcrit. Reject H₀ when |tobserved| ≥ tcrit.
  4. Mind the bottom row. At df = ∞ the t-values become the z-values — the normal table is just t with infinite degrees of freedom.
When to use
  • Any test of means where the population variance is unknown.
  • Choosing a critical value from α (one- or two-tailed) and df.
  • Small samples, where the t-correction matters most.
